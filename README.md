**Potato Disease Classification**

Potato Disease Classification is a smart web and mobile application designed to assist farmers in diagnosing potato plant diseases through leaf image analysis. By leveraging Convolutional Neural Networks (CNNs), the system can accurately detect and classify common potato leaf diseases, helping farmers take timely and informed actions to protect their crops.

**How It Works:**
Users can simply upload an image of a potato leaf through the app or website. The image is then processed using a trained deep learning model built with TensorFlow, which classifies the disease and provides insights on the identified issue.
